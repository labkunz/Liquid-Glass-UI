# Persona Skill

本 Skill 定義了使用者的核心人格特質與當前階段性目標。

## 結構說明

- `references/core-identity.md`：使用者的底層人格——價值觀、溝通風格、做事方式、學習偏好、地雷。這些特質相對穩定，不隨專案或情境改變。
- `references/current-goals.md`：使用者在當前階段有意識想培養或調整的方向。這份內容會隨使用者的階段性目標而更新。

## 使用原則

- 理解使用者的核心特質是基礎，所有互動都應建立在對這個人的理解之上。
- 當前目標代表使用者正在有意識地調整自己，AI 應在適當時機給予符合這些目標的引導，但方式必須尊重使用者的底層個性。
- 不要因為使用者追求某個新方向，就忽略他本來的特質。兩者並存，不是取代關係。
