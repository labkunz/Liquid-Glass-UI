# 核心人格特質

## 價值觀

這個人的核心驅動力是「認真」。他重視品質、重視邏輯、重視每一次互動是否經過思考。對他來說，敷衍不只是能力問題，是態度問題。

他天生是完美主義者，傾向把事情做到位才放手。這不是缺點，而是他對品質的堅持，但他自己也意識到這個特質有時會成為行動的阻力。

## 做事方式

他需要方向感才能安心行動。拿到任何任務，第一步一定是確認「我到底要做什麼」——沒有方向感對他來說是最不舒服的狀態。

他預設追求品質，但有明確的切換條件：當某件事的在意程度高到佔據日常七成以上的心智空間時，會切換成速度優先。這不是他天生求快，而是急迫感驅動的模式切換。

面對不確定性，他會盡量收集資訊到有信心才行動。這代表在他提出模糊需求時，比起直接給答案，更好的做法是先幫他釐清方向。

## 溝通風格

他溫和但重論述，會根據對象調整表達方式。他的邏輯性很強，講話有結構感，同時也會顧及對方的感受。

他對溝通品質的標準很高。以下是他的核心地雷——不是語氣問題，而是態度與思考品質的問題：

- 敷衍了事、不認真對待他的需求
- 表面漂亮但經不起邏輯檢驗的回應
- 沒搞清楚他在問什麼就急著給答案
- 過度樂觀或用雞湯式的話語迴避問題

他判斷一次互動好不好，最核心的標準只有一個：對方有沒有真的聽懂他想嘗試什麼。

## 學習方式

他目前偏好「邊做邊查」的學習節奏——先動手，遇到問題再補知識。如果這條路行不通，會轉向看範例或官方文件來反推邏輯、確認理解。

這代表在學習場景中，不需要先給他一大段理論，而是陪他在實作過程中按需補充。但當他主動問「為什麼」的時候，要能給出有深度的解釋，而不是敷衍帶過。
