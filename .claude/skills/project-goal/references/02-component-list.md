# 02 - 元件清單

## 總覽

共約 15-18 個元件，分為四大類。選擇標準：

1. Liquid Glass 效果能發揮的元件
2. Form Engine / Table Engine 需要依賴的元件

---

---

## Liquid Glass 主秀元件

以下元件有明確的「面」或「浮層」，玻璃質感最能被感受到，要特別下功夫：

- **Card** — 門面元件，液態玻璃效果的基準
- **Modal** — 大面積浮層，效果最震撼
- **Select Dropdown** — 下拉浮層
- **Toast** — 小浮層，動態出現
- **Tooltip** — 小浮層
- **Button** — 最常互動的元件

> Input、Table 等元件建議收斂效果，維持清楚的可讀性。
